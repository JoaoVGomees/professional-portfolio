Relógio Digital

O relógio digital é uma aplicação web simples que exibe a hora atual do sistema em formato digital. Funciona através de um script JavaScript que atualiza constantemente as horas, minutos e segundos exibidos na interface do usuário. Os estilos visuais são definidos no arquivo CSS. Ao abrir a página HTML, o script obtém a hora do sistema e a exibe dinamicamente.
